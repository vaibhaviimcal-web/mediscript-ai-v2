# MediScript AI v2 - Enterprise Prescription System

## Real AI-Powered Prescription Generation

Built with:
- **Groq API** (Llama 3.1 70B) - FREE AI
- **Web Speech API** - Voice input
- **jsPDF** - PDF generation
- **Tailwind CSS** - Modern UI

## Setup Instructions

1. Clone this repository
2. Add your Groq API key to `config.js`
3. Open `index.html` in browser or deploy to GitHub Pages

## Features

✅ Patient Management (CRUD)
✅ Appointment Scheduling
✅ AI Prescription Generator
✅ PDF Generation
✅ Dashboard with Analytics
✅ Mobile Responsive
